Docker部署一个spa应用

用 vue cli 创建一个 vue 项目，修改一下创建出来的项目，在页面上写一个前端接口请求，构建一版线上资源 ，基于 nginx docker 镜像构建成一个前端工程镜像，然后基于这个前端工程镜像，启动一个容器 vuenginxcontainer。启动一个基于 node 镜像的容器 nodewebserver，提供后端接口。修改 vuenginxcontainer 的 nginx 配置，使前端页面的接口请求转发到 nodewebserver 上。稍作优化和改进。
