滚动条禁用边缘回弹

2017-09-12
IM, browser

webkit内核浏览器比如chrome、safari在处理页面滚动操作时会默认增加一个边缘回弹效果，本意是让人机交互更流畅，不至于出现界面突然静止的情况。但这个效果在electron页面里面却不太好，因为这样的话把鼠标放在窗口里滚动，即使上下没有滚动的内容，web窗口也会跟着上下抖，显得整个应用不那么“原生”。
